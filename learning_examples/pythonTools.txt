1. Download anaconda for Linux "anaconda.sh"
2. Add the execution permision for the file by run: $ chmod +x anaconda.sh
3. Install: $ ./anaconda.sh
4. after installation, 
	run anaconda navigator: $ anaconda-navigator
	run jupiter: 			$ ipython notebook or jupyter-notebook
	run spyder:				$ spyder

|pycharm 	  |
+-------------+
1. download pycharm for Linux "pycharm-community-2017.2.3.tar.gz"
2. unzip the compressed file: $ tar xvf pycharm-community-2017.2.3.tar.gz
3. go to the installation directory: cd cd pycharm-community-2017.2.3/bin
4. Add the execution permision for the file by run: $ chmod +x pycharm.sh
5. run: ./pycharm.sh
